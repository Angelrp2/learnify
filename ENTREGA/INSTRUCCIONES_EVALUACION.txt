INSTRUCCIONES DE EVALUACIÓN - LEARNIFY
=======================================

TIEMPO ESTIMADO: 45-60 minutos

PASO 1: REVISAR LA MEMORIA TÉCNICA

Abre MEMORIA_TECNICA.pdf y revisa:
- Sección 2: Arquitectura elegida
- Sección 3: Conceptos JavaScript aplicados (con ejemplos de código)
- Sección 4: Sistema de estilos (variables :root + Tailwind)
- Sección 5: Estrategia SEO aplicada

PASO 2: ACCEDER AL REPOSITORIO

Ve a https://github.com/Angelrp2/learnify

Revisa:
- README.md — introducción e instalación
- /docs/ — documentación técnica (7 archivos)
- /astro/src/ — código del frontend
- /strapi/src/ — colecciones del backend
- Historial de commits — proceso de desarrollo

PASO 3: EJECUTAR STRAPI

Terminal 1:
  git clone https://github.com/Angelrp2/learnify.git
  cd learnify/strapi
  pnpm install     (o npm install si hay errores SSL)
  cp .env.example .env
  pnpm run dev

Accede a http://localhost:1337/admin y crea cuenta de administrador.

IMPORTANTE — configurar permisos públicos:
  Settings > Roles > Public
  Habilitar find/findOne para: Categories, Courses, Instructors,
  Lessons, Blog-posts
  Habilitar create para: Contact-submissions
  Guardar

Crear contenido de prueba según STRAPI_ACCESO.txt

PASO 4: EJECUTAR ASTRO

Terminal 2:
  cd learnify/astro
  pnpm install
  cp .env.example .env
  pnpm run dev

Accede a http://localhost:4321

PASO 5: VERIFICAR FUNCIONALIDADES

PÁGINA DE INICIO (localhost:4321):
✓ Hero section con botones
✓ Cursos destacados cargados desde Strapi
✓ Sección de categorías con enlaces

CURSOS (localhost:4321/cursos):
✓ Buscador filtra en tiempo real con regex
✓ Filtros por dificultad funcionan
✓ Filtro por categoría funciona
✓ Ordenación cambia el orden
✓ Paginación navega entre páginas
✓ Cards con imagen, badge de dificultad y enlace

CATEGORÍAS (localhost:4321/categorias):
✓ Lista todas las categorías con número de cursos
✓ Al hacer clic muestra los cursos de esa categoría

CURSO INDIVIDUAL (localhost:4321/cursos/[slug]):
✓ Título, descripción, dificultad, duración
✓ Instructor y categoría

BLOG (localhost:4321/blog):
✓ Artículos con tiempo de lectura calculado automáticamente
✓ Fecha de publicación

CONTACTO (localhost:4321/contacto):
✓ Enviar formulario vacío muestra errores en rojo
✓ Email inválido muestra error específico
✓ Checkbox RGPD obligatorio
✓ Estado loading en el botón al enviar
✓ Mensaje de éxito al enviar correctamente

DARK MODE:
✓ Botón luna/sol en el navbar (arriba derecha)
✓ Persiste al recargar la página

RESPONSIVE:
✓ F12 > vista móvil 375px — diseño se adapta

SEO TÉCNICO:
✓ localhost:4321/sitemap.xml — incluye URLs de cursos y blog
✓ localhost:4321/robots.txt — bloquea /admin y /api
