LEARNIFY - ENTREGA PROYECTO INTEGRADOR
======================================

CONTENIDOS DE LA ENTREGA:

1. MEMORIA_TECNICA.pdf
   - Arquitectura elegida (Astro + Strapi)
   - Aplicación de contenidos técnicos (JavaScript avanzado)
   - Sistema de estilos (CSS variables + Tailwind)
   - Estrategia SEO aplicada (on-page y técnico)

2. GITHUB_REPOSITORY.txt
   - URL del repositorio
   - Instrucciones de acceso

3. STRAPI_ACCESO.txt
   - Instrucciones para acceder a Strapi localmente
   - Pasos de instalación

4. INSTRUCCIONES_EVALUACION.txt
   - Cómo evaluar el proyecto
   - Qué verificar

REQUISITOS CUMPLIDOS:

✓ Arquitectura clara (frontend/backend separados)
✓ Control de versiones con Git
✓ Frontend: HTML5, CSS responsive, JavaScript avanzado
✓ Validación de formularios
✓ Comunicación con backend vía fetch
✓ SEO técnico completo
✓ Contenido relevante y humanizado
✓ Dark mode y UX profesional
✓ API externa integrada (GitHub API)
✓ Manejo de errores completo
✓ Backend: Node.js (Strapi), API REST
✓ Base de datos relacional (6 tablas)
✓ Diagrama E-R documentado
✓ CRUD funcional
✓ 4 roles de usuario
✓ Seguridad: validación, protección XSS, RGPD
✓ Documentación completa

CONCEPTOS JAVASCRIPT IMPLEMENTADOS:

✓ Desestructuración
✓ Array methods (map, filter, sort, slice)
✓ Async/await
✓ Try/catch
✓ Expresiones regulares
✓ Normalización de texto
✓ Validación de formularios
✓ Cálculo de tiempo de lectura
✓ Paginación

FUNCIONALIDADES EXTRAS:

✓ Búsqueda con regex
✓ Filtros dinámicos
✓ Paginación inteligente
✓ Dark/light mode
✓ API externa (GitHub)
✓ Arquitectura SSG (mejor rendimiento)

PRÓXIMOS PASOS DESPUÉS DE LA EVALUACIÓN:

1. Desplegar Strapi en Render.com
2. Desplegar Astro en Netlify.com
3. Agregar URL desplegada al PDF
4. Compartir URLs finales
