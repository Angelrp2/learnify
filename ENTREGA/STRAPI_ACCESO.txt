ACCESO FUNCIONAL A STRAPI - LEARNIFY
=====================================

INSTALACIÓN Y EJECUCIÓN LOCAL:

1. Clonar el repositorio:
   git clone https://github.com/Angelrp2/learnify.git
   cd learnify

2. Instalar dependencias de Strapi:
   cd strapi
   pnpm install
   (Si pnpm falla con errores SSL, usa: npm install)

3. Copiar archivo de configuración:
   cp .env.example .env

4. Iniciar Strapi:
   pnpm run dev
   (o: npm run dev)

5. Acceder al panel de administración:
   http://localhost:1337/admin

PRIMERA EJECUCIÓN:

En la primera ejecución Strapi pedirá crear una cuenta de administrador.
Completa el formulario con email, contraseña y nombre.

CONFIGURAR PERMISOS PÚBLICOS (obligatorio):

1. Ve a Settings > Roles > Public
2. Habilita "find" y "findOne" para:
   - Categories
   - Courses
   - Instructors
   - Lessons
   - Blog-posts
3. Habilita "create" para Contact-submissions
4. Guarda los cambios

Sin este paso la API no devuelve datos y el frontend aparece vacío.
